# My Kipawa

## Overview
A full-stack platform for showcasing talent, selling merchandise, and sharing event highlights.

## Tech Stack
- Backend: Java (Spring Boot)
- Frontend: React
- Database: PostgreSQL

## Features
- User authentication
- Media highlights
- Merchandise store
- Payments (M-Pesa, Card, PayPal)

## Project Structure
- mykipawa-backend/
- mykipawa-frontend/
